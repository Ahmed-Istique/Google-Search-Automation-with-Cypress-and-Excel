**Project Overview** :

A JavaScript-based automation system that performs Google searches and captures search suggestions using Cypress.

**Programming Language** : JavaScript (Node.js environment)

**Technologies & Frameworks**


1 . Cypress - End-to-end testing framework


2 . xlsx (SheetJS) - Excel file manipulation library


3 . Excel/XLSX - Data storage and results tracking


**Workflow**

JavaScript → Cypress → Google Search → Excel Data Processing → Results Storage
